# Blog

This is my personal website and blog built using Hugo.

Live at: https://bksahu.me

1. Installation (For Arch Linux)

```console
$ sudo pacman -S hugo 
```

2. Running dev server
```console
$ hugo serve -D
```

3. Add a new blogpost
```console
$ hugo new blog/<post-name>.md
```

4. Deploying to github
```console
$ sh deploy.sh
```
